# ROOT/tech_design/go_module

## Intent

Documents the Go module configuration and how it is managed.

## Contracts

The module path is:
`github.com/CodeFromSpec/tool-subagent-mcp`

## Constraints

- `go.mod` is not generated by a spec node. It is managed by Go
  tooling.
- After code generation, run `go mod tidy` to update dependencies
  based on the imports in generated code.
- `go.sum` is generated automatically by Go tooling — never edit
  it manually.

## Context

The module is already initialized in this repository. After code
generation, run:

```
go mod tidy
```

to pull `github.com/mark3labs/mcp-go` and `gopkg.in/yaml.v3`.
